## 六、数据获取方法

### 6.1 使用Python SDK获取详细数据

```python
from cloudpss import Model

# 获取模型
model = Model.fetch('model/holdme/test_ieee39_wind')

# 运行潮流
job = model.runPowerFlow()
job.wait()
result = job.result

# 1. 获取所有节点数据
buses = result.getBuses()
for bus in buses:
    print(f"{bus['name']}: V={bus['voltage']:.4f} p.u., "
          f"Gen={bus.get('generation', {}).get('P', 0):.2f} MW")

# 2. 获取所有支路数据
branches = result.getBranches()
for branch in branches:
    print(f"{branch['from']}->{branch['to']}: "
          f"P={branch.get('P_from', 0):.2f} MW")

# 3. 统计信息
total_gen = sum(b.get('generation', {}).get('P', 0) for b in buses)
total_load = sum(b.get('load', {}).get('P', 0) for b in buses)
print(f"总发电: {total_gen:.2f} MW")
print(f"总负荷: {total_load:.2f} MW")
print(f"网损: {total_gen - total_load:.2f} MW")
```
