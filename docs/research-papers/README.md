# 研究论文目录

此目录用于存放待技能化的学术论文。

## 目录结构

```
research-papers/
├── README.md                    # 本文件
├── lvrt_short_circuit/         # 新能源LVRT短路电流分析
│   ├── README.md              # 论文摘要和关键点
│   └── [paper_files]          # 原始论文 PDF
├── voltage_stability/          # 电压稳定分析
│   └── ...
└── ...
```

## 上传论文流程

1. 将论文 PDF 放入对应主题目录
2. 创建 README.md 记录：
   - 论文标题、作者、发表年份
   - 核心算法描述
   - 关键参数
   - 适用场景
3. 提交到仓库

## 技能化流程

1. Agent 研读论文
2. 提取算法要点
3. 设计技能接口
4. 实现并验证
5. 合并到技能库
